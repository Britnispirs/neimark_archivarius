# AI-Archivarius

**AI-Archivarius** — это асинхронный Telegram-бот на базе RAG-архитектуры (Retrieval-Augmented Generation). Он позволяет загружать объемные текстовые документы, лекции или книги, а затем задавать по ним вопросы. Бот находит нужные фрагменты в тексте и с помощью ИИ формулирует точный ответ.

## Фичи проекта

- **Мультиформатность:** Поддержка чтения `.txt`, `.pdf` и `.docx`.
- **Локальная векторная база данных:** Использование `ChromaDB` для быстрого семантического поиска.
- **Обход лимитов ИИ:** Внедрен алгоритм жесткого чанкования (нарезка текста по абзацам), что полностью решает проблему переполнения окна контекста нейросети.
- **Асинхронность:** Бот написан на `aiogram 3.x` и не зависает при работе с несколькими юзерами.

## Стек технологий

- Python 3.12+ 
- Aiogram 3
- ChromaDB
- Mistral AI (SDK v2.5.0)
- Sentence-Transformers (модель `all-MiniLM-L6-v2`)
- `pypdf`, `python-docx`

## Как запустить у себя

**1. Клонируем репозиторий**
`git clone https://github.com/ВАШ_НИК/ai-archivarius.git`
`cd ai-archivarius`

**2. Создаем окружение и ставим зависимости**
`python3 -m venv venv`
`source venv/bin/activate`
`pip install -r requirements.txt`

**3. Настраиваем ключи**
Создайте файл `.env` в корне проекта и впишите туда токены:
`BOT_TOKEN=токен_от_телеграма`
`MISTRAL_API_KEY=токен_от_мистраля`

**4. Запускаем**
`python3 main.py`

## Как пользоваться

1. Жмем `/start` в боте.
2. Кидаем любой документ (`.txt`, `.pdf` или `.docx`).
3. Бот скажет, что всё изучил.
4. Пишем любой вопрос по тексту. Бот ответит, опираясь только на загруженный файл.
